# 素材を入れる場所

ここは公開用の素材フォルダです。

まずは下の場所に画像や音声を入れてください。
入れ終わったら「入れたよ」と言ってくれれば、こちらでファイル名を見て `asset-map.js` に対応を書きます。

- 画像: `kinbaku-clicker-assets/images/`
- BGM: `kinbaku-clicker-assets/audio/`
- ボイス: `kinbaku-clicker-assets/audio/voices/`
- 効果音: `kinbaku-clicker-assets/audio/sfx/`

今は `asset-map.js` を自分で編集しなくて大丈夫です。
素材を入れ終わったあと、こちらでまとめて設定します。

GitHub Pagesで公開する時は、`index.html`、`kinbaku-clicker.html`、`kinbaku-clicker-assets` フォルダをまとめてアップロードします。
